# 游戏截图工具 - 使用说明

## 打包成功！

**EXE 文件位置**: `dist/game_capture.exe`
**文件大小**: 约 27MB

---

## 使用步骤

### 1. 运行程序
双击 `game_capture.exe` 或在命令行运行：
```
game_capture.exe
```

### 2. 输入游戏窗口名称
程序会提示你输入游戏窗口的标题栏名称。

**如何查找窗口名称**：
- 查看游戏窗口顶部的标题栏
- 或使用附带工具查看：
  ```
  python -c "import win32gui; [print(win32gui.GetWindowText(h)) for h in win32gui.EnumWindows() if win32gui.IsWindowVisible(h)]"
  ```

### 3. 配置截图参数
- **输出目录**: 默认为 `images`
- **截图间隔**: 默认为 0.3 秒

### 4. 停止截图
按 `Ctrl+C` 停止

## 规避检测的测试建议

### 测试步骤：
1. **先单独测试 EXE**：
   - 打开游戏
   - 运行 `game_capture.exe`
   - 观察是否被检测

2. **如果仍被检测**，可能的原因：
   - 检测机制不仅是进程名，可能包括：
     - 内存扫描
     - 行为检测（截图频率）
     - 驱动级检测

3. **进一步规避思路**：
   - 更改 EXE 名称和图标
   - 使用 UPX 压缩（已启用）
   - 添加代码混淆
   - 使用虚拟机/沙箱检测绕过
